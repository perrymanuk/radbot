# RadBot Agent Workflow Guides

Procedural guides for AI agents working on this codebase.
Read these before making changes — they explain how the pieces fit together
and the steps to follow for common tasks.

## Maintenance
| Guide | When to Use |
|-------|------------|
| [maintaining-agents.md](maintaining-agents.md) | Updating agent instructions, tools, or routing |
| [maintaining-integrations.md](maintaining-integrations.md) | Fixing or updating external service clients |
| [maintaining-frontend.md](maintaining-frontend.md) | React SPA changes, new panels, API client updates |
| [maintaining-db.md](maintaining-db.md) | Schema changes, migrations, new tables |
